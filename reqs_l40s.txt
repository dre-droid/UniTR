# 1. Core PyTorch & Build Tools (Run via Conda)
# - pytorch-cuda=12.1: Matches the L40S hardware (Lovelace architecture).
# - cuda-nvcc=12.1: Installs the C++ compiler required to build custom 3D extensions via setup.py.
# - mkl<2024 / intel-openmp<2024: Prevents the PyTorch 2.5 "iJIT_NotifyEvent" CPU crash bug.
# - gcc=11 gxx=11: Downgrade needed by CUDA12.1
# - cuda-cccl=12.1: Fix for compilation errors
# Command: conda install pytorch torchvision torchaudio pytorch-cuda=12.1 cuda-nvcc=12.1 nvidia cuda-toolkit=12.1 "mkl<2024" "intel-openmp<2024" -c pytorch -c nvidia -y
# Command: conda install gcc=11 gxx=11 python=3.10 -c conda-forge -y
# Command: conda install -c nvidia cuda-cccl=12.1 -y
# Command: conda install -c nvidia cuda-libraries-dev=12.1 -y
